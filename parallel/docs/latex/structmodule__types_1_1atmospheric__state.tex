\doxysubsection{Detailed Description}
Derived type for the time-\/dependent atmospheric state variables. 

This type holds the core prognostic variables ({\ttfamily dens}, {\ttfamily umom}, {\ttfamily wmom}, {\ttfamily rhot}) and includes halo regions ({\ttfamily hs}) for parallel exchange.
