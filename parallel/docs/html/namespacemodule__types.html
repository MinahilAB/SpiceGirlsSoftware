◆ exchange_halo_x()

subroutine module_types::exchange_halo_x ( class(atmospheric_state), intent(inout)  s)
private

Performs the MPI-based halo exchange in the X-dimension.

This routine is responsible for point-to-point communication with neighboring ranks. It involves packing data, initiating non-blocking sends/receives, and waiting for completion. The wall-clock time is captured by the NVTX range :exchange_halo_x.

Parameters
[in,out]sThe atmospheric state whose halo regions need updating.

class(reference_state), intent(in)  ref 
)
private

Implements the Z-dimension (vertical) halo exchange.

Since the Z-dimension is not parallelized, this involves applying local, non-communicative boundary conditions (e.g., free slip, fixed values) to the halo regions. The wall-clock time is captured by the NVTX range :exchange_halo_z.

Parameters
[in,out]sThe atmospheric state to apply boundary conditions to.
[in]refThe reference state used to calculate some boundary values.

)
private

Calculates the X-dimension contribution to the total tendency.

This routine involves two main steps: flux calculation (using a 4-point stencil) and tendency calculation (flux difference). It requires an X-halo exchange beforehand. The wall-clock time is captured by the NVTX range :xtend.

Parameters
[in,out]tendencyThe resulting time tendency array.
[in,out]fluxTemporary flux array used for calculation.
[in]refReference state data.
[in,out]atmostatCurrent atmospheric state (required for halo data).
[in]dxGrid spacing in the X-direction.
[in]dtTime step size (used for hyperviscosity coefficient).

)
private

Calculates the Z-dimension contribution to the total tendency.

This routine involves two main steps: flux calculation (using a 4-point stencil) and tendency calculation (flux difference), plus the application of gravity as a source term. It requires a Z-halo exchange beforehand (local boundary condition setting). The wall-clock time is captured by the NVTX range :ztend.

Parameters
[in,out]tendencyThe resulting time tendency array.
[in,out]fluxTemporary flux array used for calculation.
[in]refReference state data.
[in,out]atmostatCurrent atmospheric state (required for halo data).
[in]dzGrid spacing in the Z-direction.
[in]dtTime step size (used for hyperviscosity coefficient).
