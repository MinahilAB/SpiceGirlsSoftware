Detailed Description

Defines all core derived types for the atmospheric model state, fluxes, and tendencies, along with parallel data management and kernel execution routines.

This module establishes the data structures and core subroutines (like xtend, ztend, and halo exchange) that drive the 3D stencil computation. It is crucial for understanding the data layout and the structure of the time integration step.
